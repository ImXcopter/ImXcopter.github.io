# 发光二极管正负极怎么分？

发光二极管一般有以下 2 种：直插型 LED 发光二极管、贴片 LED 发光二极管。
**直插型 LED 发光二极管判断正负极**
方法 1：用观眼睛察引脚长短也可以看出发光二极管的正负极，引脚长的为正极，短的为负极。
![](/static/2023/2023-03-02-led-polarity-guide_001.jpg)
方法 2：用眼睛观察 LED 二极管内部，支架大衔接的引脚是负极，支架小的连接的引脚是正极。
![](/static/2023/2023-03-02-led-polarity-guide_002.png)
方法 3：用指针式万用表电阻档：R×1k 档，可以用来检测发光二极管。检测时，将两表笔分别与发光二极管的两条引脚相接，如表向右偏转，并且发光二极管中有发光点，说明发光二极管是正向连接。这时与黑表笔(表内电池正极相连)相接的是正极；与红表笔(表内电池负极相连)相接的是负极。
方法 4：用数字万用表，将旋钮打到通断档，将红黑表笔分别接在两个引脚。如果有读数，则红表笔一端为正极，黑表笔一端为负极；若读数为“1”，则黑表笔一端为正极，红表笔一端为负极。
**贴片LED发光二极管判断正负极**
1.5050贴片LED是正方形的，四个直角中，其中有一个角是小缺角，其他的直角没有小缺角，带小缺角的那端就是负极，另一端则是正极。
![](/static/2023/2023-03-02-led-polarity-guide_003.png)
2. 0805、0603封装的贴片发光二极管在底部有倒三角形符号，三角形符号的“边”靠近的是正，“角”靠近的是负极。
![](/static/2023/2023-03-02-led-polarity-guide_004.png)
3. 贴片发光二极管在底部都会有“T”字形，“T”一横的一边是正极，另一边是负极。
![](/static/2023/2023-03-02-led-polarity-guide_005.png)
