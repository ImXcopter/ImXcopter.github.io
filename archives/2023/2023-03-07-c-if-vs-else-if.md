# C 语言中，if... 和 if...else if 的区别是什么？

**if...else 和 else if 在结构上的区别：**

**if...else 结构：**

```c
if (条件表达式) {
    执行语句1
} else {
    执行语句2
}
```

如果条件表达式为真，则执行语句1。如果条件表达式为假，则执行语句2。

![if...else 结构图](/static/2023/2023-03-07-c-if-vs-else-if_001.png)

**else if 结构：**

```c
if (条件表达式1) {
    执行语句1
} else if (条件表达式2) {
    执行语句2
} ...
} else if (条件表达式n-1) {
    执行语句n-1
} else {
    执行语句n
}
```

如果条件表达式1成立，则执行语句1；如果条件表达式1不成立，然后条件表达式2成立，则执行语句2；如果所有条件表达式都不成立，则执行语句n。

![else if 结构图](/static/2023/2023-03-07-c-if-vs-else-if_002.png)

## 总结

| 特性 | if...else | else if |
|------|-----------|---------|
| 条件数量 | 2 个 | 多个 |
| 执行逻辑 | 二选一 | 多选一（按顺序判断） |
| 特点 | 条件互斥 | 前面的条件不成立才判断下一个 |
