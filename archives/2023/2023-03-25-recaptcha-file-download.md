# Recaptcha to Download a File HTML 代码

本教程展示了如何整合 reCaptcha 来下载一个文件。这是 HTML 例子，对所有编程语言都适用。

![](/static/2023/2023-03-25-recaptcha-file-download_001.jpg)

请注意，这个例子将在更新 reCAPTCHA 网站的信息后在服务器中运行。

## 步骤 1：获取网站密钥

从 Google reCAPTCHA 网站获得网站密钥。请访问下面的链接，了解详细的步骤过程。

> 参考教程：https://chillyfacts.com/add-recaptcha-i-m-not-a-robot-in-java-websites/

![](/static/2023/2023-03-25-recaptcha-file-download_002.jpg)

## 步骤 2：创建 HTML 文件

创建 `test.html` 文件：

```html
<!-- Click to download the file -->
```

访问地址：`http://localhost:8080/Test_Rec/test.html`

![](/static/2023/2023-03-25-recaptcha-file-download_003.jpg)
