# 微软Azure API申请教程

## 准备工作

开始之前，需要先申请[即用即付](https://azure.microsoft.com/zh-cn/pricing/purchase-options/pay-as-you-go/)服务

![申请即用即付服务](/static/2023/2023-12-18-azure-api-application-guide_001.png)

按照提示登录微软账号、填写个人资料、付款信息即可，这里不再详述。过程中需要用到信用卡，会扣费1$, 以后只要不超免费额度就不会扣费。

![付款信息](/static/2023/2023-12-18-azure-api-application-guide_002.png)

## 1. 登录 Microsoft Azure

访问：[Microsoft Azure](https://portal.azure.com/)

### 点击"创建资源"

![创建资源](/static/2023/2023-12-18-azure-api-application-guide_003.png)

## 2. 点击"AI+机器学习"，创建"语音"服务

![创建语音服务](/static/2023/2023-12-18-azure-api-application-guide_004.png)

## 3. 根据提示填写完毕后，点击"审阅并创建"

![审阅并创建](/static/2023/2023-12-18-azure-api-application-guide_005.png)

## 4. 验证通过后，继续点击"创建"

![创建服务](/static/2023/2023-12-18-azure-api-application-guide_006.png)

## 5. 等待部署完成，点击"转到资源"

![转到资源](/static/2023/2023-12-18-azure-api-application-guide_007.png)

## 6. 可以看到"密钥"和"区域"

![密钥和区域](/static/2023/2023-12-18-azure-api-application-guide_008.png)
