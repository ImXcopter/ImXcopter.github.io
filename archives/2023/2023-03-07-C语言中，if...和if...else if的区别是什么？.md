**if...else和else if在结构上的区别：**
if （条件表达式）+ 执行语句1
else + 执行语句2
如果条件表达式为真，则执行语句1。如果条件表达式为假，则执行语句2。
![](/static/2023/2023-03-07-c-if-vs-else-if_001.png)
if （条件表达式1）+ 执行语句1
elseif （条件表达式2）+ 执行语句2
......
elseif（条件表达式n-1）+ 执行语句n-1
else + 执行语句n
如果条件表达式1成立，则执行语句1，如果条件表达式1不成立，然后条件表达式2成立，则执行语句2，如果所有条件表达式都不成立，则执行执行语句n。
![](/static/2023/2023-03-07-c-if-vs-else-if_002.png)
