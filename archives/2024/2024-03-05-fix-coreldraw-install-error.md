# CorelDRAW Graphics Suite 安装提示：A newer version of this application is already installed.Installation stopped. 解决办法

![image.png](/static/2024/2024-03-05-fix-coreldraw-install-error_001.png "1642490811722954.png")

问题原因：
-----

Vba71组件无法安装或跟SolidWorks软件的插件发生冲突

解决方案：
-----

方案1：重新安装，选择自定义安装-实用工具-取消勾选Visual Basic for Application7.1后继续安装即可
![image.png](/static/2024/2024-03-05-fix-coreldraw-install-error_002.png "1642490863677719.png")
方案2：打开控制面板卸载Visual Basic for Application7.1或卸载SolidWorks软件后重新安装CorelDRAW
（若控制面板中没有Visual Basic for Application7.1可用清理工具进行查找卸载清理）
